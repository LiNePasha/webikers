# Bundle Page - دليل الاستخدام

## 📦 نظرة عامة
صفحة Bundle تسمح للمستخدمين بشراء مجموعة من المنتجات مع بعض في صفقة واحدة، مع دعم كامل للمنتجات المتغيرة (Variable Products) **والكميات المتعددة**.

## 🔗 صيغة الرابط

### 1️⃣ منتجات عادية (Simple Products)
```
/bundle?p=123,456,789
```
- `123`, `456`, `789` = Product IDs (quantity = 1 by default)

### 2️⃣ منتجات عادية بكميات محددة
```
/bundle?p=123:3,456:2,789:5
```
- `123:3` = منتج 123، كمية 3
- `456:2` = منتج 456، كمية 2
- `789:5` = منتج 789، كمية 5

### 3️⃣ منتجات متغيرة (Variable Products)
```
/bundle?p=36528:42657,36529:42658
```
- `36528:42657` = Product ID : Variation ID (quantity = 1)
- `36529:42658` = Product ID : Variation ID (quantity = 1)

### 4️⃣ منتجات متغيرة بكميات محددة
```
/bundle?p=36528:42657:2,36529:42658:3
```
- `36528:42657:2` = Product 36528, Variation 42657, كمية 2
- `36529:42658:3` = Product 36529, Variation 42658, كمية 3

### 5️⃣ خليط من كل الأنواع
```
/bundle?p=123:3,36528:42657:2,789,36529:42658
```
- `123:3` = منتج عادي، كمية 3
- `36528:42657:2` = منتج متغير، كمية 2
- `789` = منتج عادي، كمية 1 (default)
- `36529:42658` = منتج متغير، كمية 1 (default)

## ⚙️ كيف يتم التمييز؟

النظام ذكي ويميز تلقائياً بين:
- **Simple + Quantity**: `123:3` (الرقم الثاني < 100 → كمية)
- **Variable (no quantity)**: `36528:42657` (الرقم الثاني > 100 → variation ID)
- **Variable + Quantity**: `36528:42657:2` (3 أرقام → product:variation:quantity)

> **ملحوظة**: Variation IDs عادة أرقام كبيرة (> 1000)، بينما الكميات أرقام صغيرة (1-99)

## ✨ المميزات

### 🎯 دعم المنتجات المتغيرة
- يقرأ الـ variation ID من URL
- يجلب بيانات الـ variation من WooCommerce
- يعرض attributes المحددة (مثل: اللون، المقاس)
- يحفظ الـ variation ID في السلة

### � دعم الكميات المتعددة
- تحديد كمية مختلفة لكل منتج
- حساب تلقائي للأسعار الإجمالية (سعر × كمية)
- عرض الكمية في badge على المنتج
- إضافة الكمية المحددة للسلة مباشرةً

### 💰 حساب الأسعار
- يحسب السعر الأصلي لكل منتج × الكمية
- يحسب السعر بعد الخصم × الكمية
- يعرض نسبة التوفير الإجمالية
- يدعم أسعار الـ variations

### 🛒 إضافة للسلة
- يضيف كل المنتجات بضغطة واحدة
- يحفظ الـ variation_id للمنتجات المتغيرة
- يضيف الكمية المحددة لكل منتج
- يعرض الـ attributes في السلة
- يستخدم صور الـ variation إن وجدت

### 🎨 UI محسّن
- عرض شارات الخصم لكل منتج
- عرض شارة الكمية (×3) إذا أكثر من 1
- عرض الـ attributes تحت اسم المنتج
- عرض السعر الإجمالي (سعر الوحدة × الكمية)
- إمكانية إزالة منتجات من الحزمة
- Animations سلسة للـ products

## 📝 أمثلة عملية

### مثال 1: حزمة أحذية بكميات (3 أزواج مختلفة)
```
/bundle?p=36528:42657:2,36528:42658,36528:42659:3

حيث:
- 36528:42657:2 = حذاء Nike مقاس 42 - كمية 2
- 36528:42658 = حذاء Nike مقاس 43 - كمية 1 (default)
- 36528:42659:3 = حذاء Nike مقاس 44 - كمية 3
الإجمالي: 6 أحذية
```

### مثال 2: باقة قطع غيار للورشة
```
/bundle?p=123:10,456:999:4,456:1000:4,789:2

حيث:
- 123:10 = فلتر زيت - 10 قطع
- 456:999:4 = إطار 185/65R15 - 4 إطارات
- 456:1000:4 = إطار 195/65R15 - 4 إطارات
- 789:2 = زيت محرك - 2 عبوة
```

### مثال 3: عرض ملابس (نفس المنتج، ألوان وكميات مختلفة)
```
/bundle?p=2500:3001:5,2500:3002:3,2500:3003:2

حيث:
- 2500:3001:5 = تيشيرت أبيض - 5 قطع
- 2500:3002:3 = تيشيرت أسود - 3 قطع
- 2500:3003:2 = تيشيرت أزرق - 2 قطع
الإجمالي: 10 تيشيرتات
```

### مثال 4: عرض مخلوط (simple + variable + quantities)
```
/bundle?p=phone:black:1,case:red:2,charger:3,screen:5

باقة موبايل + إكسسوارات:
- phone:black:1 = موبايل أسود - 1 جهاز
- case:red:2 = جراب أحمر - 2 قطعة
- charger:3 = شاحن سريع - 3 شواحن
- screen:5 = واقي شاشة - 5 قطع
```

## 🔧 كيفية الحصول على الـ Variation ID

### الطريقة 1: من صفحة المنتج
1. افتح صفحة المنتج في المتصفح
2. اختر الـ variation المطلوب
3. شوف الـ Network tab في Developer Tools
4. ابحث عن request للـ variation
5. الـ ID هيكون في الـ response

### الطريقة 2: من WooCommerce Admin
1. افتح المنتج في WP Admin
2. اذهب لـ Variations tab
3. وسّع الـ variation المطلوب
4. الـ ID موجود في الـ URL: `post=42657`

### الطريقة 3: من API
```javascript
// Get all variations for a product
GET /api/products/36528/variations

// Response:
[
  { id: 42657, attributes: [...] },
  { id: 42658, attributes: [...] }
]
```

## ⚡ نصائح للاستخدام الأمثل

### ✅ افعل
- استخدم variation ID للمنتجات المتغيرة فقط
- تأكد إن الـ variation موجود وفي المخزون
- استخدم منتجات من نفس البائع
- اختبر الرابط قبل إرساله للعملاء

### ❌ لا تفعل
- لا تضع variation ID لمنتج عادي (simple)
- لا تخلط منتجات من بائعين مختلفين
- لا تستخدم IDs غير موجودة
- لا تنسى التأكد من توفر المنتجات

## 🎯 حالات الاستخدام

### 1. عروض الباقات
```
/bundle?p=phone:black,case:red,charger

باقة موبايل كاملة:
- موبايل - لون أسود
- جراب - لون أحمر  
- شاحن سريع
```

### 2. عروض الموسم
```
/bundle?p=shirt:xl,pants:32,shoes:42

طقم صيفي كامل:
- تيشيرت - XL
- بنطلون - مقاس 32
- حذاء - مقاس 42
```

### 3. عروض قطع الغيار
```
/bundle?p=filter,oil:5w30,brake:front

باقة صيانة دورية:
- فلتر هواء
- زيت محرك 5W-30
- فحمات فرامل أمامية
```

## ❓ استكشاف الأخطاء

### المشكلة: "لم يتم العثور على المنتجات"
- تأكد إن الـ Product IDs صحيحة
- تأكد إن المنتجات published
- تأكد من صلاحيات الـ API

### المشكلة: Variation لا يظهر
- تأكد إن الـ Variation ID صحيح
- تأكد إن الـ variation في المخزون
- تحقق من الـ Console للأخطاء

### المشكلة: السعر خطأ
- تأكد إن الـ variation لها سعر محدد
- تحقق من إعدادات الأسعار في WooCommerce
- جرب تحديث بيانات الـ product
